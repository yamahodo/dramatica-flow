# Dramatica-Flow 大模型配置指南

> 把你选择的模型配置**复制粘贴**到 API 配置页面即可，无需手动填写任何内容。

---

## 如何使用本指南

1. 打开 Dramatica-Flow 网页 → 第一步「API 配置」
2. 在下方找到你想用的模型
3. **直接复制**对应的「Base URL」和「模型名」粘贴到对应输入框
4. 填入你自己的 API Key
5. 点击保存

## 七、Ollama（本地免费，无需联网）

| 配置项 | 填写内容 |
|--------|----------|
| **Base URL** | `http://localhost:11434/v1` |
| **模型** | `qwen2.5:7b`（推荐）或 `llama3.1` 或 `deepseek-r1:8b` |
| **API Key** | **不需要填**（留空即可） |

### Ollama 安装步骤

1. 下载 Ollama：[ollama.com](https://ollama.com)
2. 安装后打开终端，输入：
   ```
   ollama run qwen2.5:7b
   ```
3. 等待模型下载完成（约 4.7GB）
4. 配置页面选「Ollama」，填上面的地址和模型名即可

> 完全免费，但需要电脑显卡好一点，否则生成速度慢。

## 八、其他兼容 OpenAI 格式的模型

只要对方提供 OpenAI 兼容接口，选「自定义」填入三项即可：

| 配置项 | 说明 |
|--------|------|
| **API Key** | 服务商给你的密钥 |
| **Base URL** | 服务商给的接口地址（通常以 `/v1` 结尾） |
| **模型** | 服务商提供的模型名称 |

常见可接入的服务：
- **硅基流动**：`https://api.siliconflow.cn/v1`，模型 `deepseek-ai/DeepSeek-V3`
- **零一万物**：`https://api.lingyiwanwu.com/v1`，模型 `yi-lightning`
- **百川智能**：`https://api.baichuan-ai.com/v1`，模型 `Baichuan4`
- **腾讯混元**：`https://api.hunyuan.cloud.tencent.com/v1`，模型 `hunyuan-pro`
- **字节豆包**：`https://ark.cn-beijing.volces.com/api/v3`，模型按实际填写
- **Groq**（国外超快）：`https://api.groq.com/openai/v1`，模型 `llama-3.1-70b-versatile`

## 常见问题

### Q：API Key 填错了会怎样？
A：会报错提示连接失败，不会扣费。检查一下 Key 是否完整复制，前后没有多余空格。

### Q：温度参数怎么填？
A：保持默认 `0.7` 就行。写小说建议 `0.7-0.9`，越高越有创意但也越不稳定。

### Q：哪个模型写小说最好？
A：推荐 DeepSeek 或通义千问 qwen-plus。免费的话用智谱 glm-4-flash 也能用，效果稍差。

### Q：可以中途换模型吗？
A：可以，随时在设置里换，已经生成的内容不会丢失。
